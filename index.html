Our Project

Right now we are in the Finishing stages of developing an unofficial Android Application for a popular local pizza chain, Gary's.

Features Now Available!

  • View Contact Page
  • Add Pizza with different sizes and toppings to an Order
  • Add Pop with different sizes and types to an Order
  • View and edit current list of items ordered
  • Enter delivery address, phone number, e-mail and payment method
  • View an order total including initial price, taxes, discounts and final price
  • Database connection to store and retrieve orders
  • Order specialty pizzas
  • View specialty pizzas ranked by ratings from customers
  • Error checking for invalid address, zip code, phone number and email
  • Edit unfulfilled orders using a unique confirmation code
  • Email Confirmation
